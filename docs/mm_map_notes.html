Flotonic's Notes on Special Banks

In the ZSCENE file, there is an 11 Command in the header. The format is as follows:

11 XX 00 00 YY 00 0Z 00

YY = Sky Box Number

Z = Disable if not 0

XX is MM-only. It says which special bank to use.

Multiply it by 8 and add 0xC58BA0 to it.

Go to that offset in the Decompressed 1.0 Version U MM rom.

The first four bytes at that offset are the start offset of the special bank.

The four bytes that follow those are the end offset of the special bank.
